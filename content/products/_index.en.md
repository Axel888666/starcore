# Products & Services

## Core Products

### Reloop Enterprise V9.1
Big data infrastructure software platform based on Apache Hadoop and Apache HBase as storage and computing cores, with Apache Ambari as the management platform.

#### Product Features
- **One-Click Installation** - GUI installation of 30+ Apache Hadoop big data ecosystem software
- **GUI Configuration** - Support GUI modification of startup scripts and background parameter configuration
- **Cluster Monitoring** - Comprehensive monitoring of component status and key indicators
- **Component Upgrade** - Quick upgrade through component address configuration
- **System High Availability** - Support HDFS, YARN, HBASE high availability
- **Cluster Security** - Kerberos+Ranger+OpenLDAP security system

#### Supported Components
- **Storage Engines**: HDFS, HBase, Kudu
- **Computing Engines**: Spark, Hive, Flink, Storm
- **Message Queues**: Kafka, Pulsar
- **Databases**: Doris, ClickHouse, Hudi
- **Management Tools**: Ambari, Ranger, Atlas
- **Monitoring Tools**: Grafana, Prometheus
