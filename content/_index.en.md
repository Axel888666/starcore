# StellerCore Technology (Shenzhen) Co., Ltd.

## Professional Big Data Infrastructure Software Platform Provider

StellerCore Technology is committed to providing reliable big data solutions and services for enterprises. Based on Apache Hadoop and Apache HBase as storage and computing cores, with Apache Ambari as the management platform, supporting Hive, Spark, Kafka, Flink, Doris, Hudi and other open source big data ecosystem software.

We provide GUI-based installation, configuration, and monitoring of various big data software to improve delivery efficiency, reduce software failures, reduce learning and maintenance costs, and overall improve enterprise big data operation efficiency.
