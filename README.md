# 星核动力科技（深圳）有限公司官网

基于Hugo静态网站生成器构建的企业门户网站，使用Event主题，支持中英文双语。

## 网站特色

- 🌐 **双语支持** - 中文（默认）和英文
- 🎨 **现代设计** - 大气美观的科技企业风格
- 📱 **响应式布局** - 完美适配各种设备
- ⚡ **高性能** - 基于Hugo的静态网站，加载速度快
- 🔧 **易于维护** - 结构清晰，内容管理简单

## 网站结构

### 主要页面
- **首页** (`/`) - 公司介绍和核心产品展示

- **产品与服务** (`/products`) - 核心产品和专业服务

### 重点解决方案

## 技术栈

- **Hugo** - 静态网站生成器

- **自定义CSS** - 企业品牌风格定制

## 本地开发

### 环境要求
- Hugo Extended版本 (>= 0.100.0)
- Git

### 快速启动
Windows用户可以直接双击运行：
- `serve.bat` - 启动开发服务器
- `build.bat` - 构建生产版本

### 手动启动
```bash
# 启动开发服务器
hugo server -D

# 构建生产版本
hugo --minify
```

### 访问网站
开发服务器启动后，在浏览器中访问：
- 中文版本：http://localhost:1313/
- 英文版本：http://localhost:1313/en/

## 内容管理

### 添加新页面
在 `content/` 目录下创建对应的markdown文件，支持中英文版本：
- 中文版本：`filename.zh.md`
- 英文版本：`filename.en.md`

### 修改配置
主要配置文件：
- `hugo.toml` - 网站基本配置
- `i18n/zh.yaml` - 中文翻译
- `i18n/en.yaml` - 英文翻译

### 自定义样式
在 `static/css/custom.css` 中添加自定义样式。
